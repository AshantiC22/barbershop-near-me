Client-Side Stories (The Customer)
As a user, I want to view a list of services (Haircut, Beard Trim) so I know what the shop offers.

As a user, I want to select a specific barber so I can book with my preferred professional.

As a user, I want to see available time slots in a calendar view so I can pick a time that fits my schedule.

As a user, I want to receive a confirmation message after booking so I know the appointment is secured.

<----------------------------------------------------->

Admin-Side Stories (The Barber)
As a barber, I want to log in to a dashboard so I can see my daily schedule.

As a barber, I want to set my "Working Hours" so clients can't book me when the shop is closed.

As a barber, I want to cancel an appointment if needed, which should automatically notify the client.
